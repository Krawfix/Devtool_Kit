# Devtool_Kit
A simple yet efficient Command Line Task Manager built with C++. This project was created as my first C++ application and focuses on implementing core programming concepts through a practical productivity tool.

## Features
- Add New tasks
- Edit Current task
- Display the stored Task 
- Delete the Task
- Set Deadline for the Task
- Mark Task as COmpleted 

## Tech Stack
- Language : C++ 
- Interface : CLI 
- Storage : Local storage ( tasks.txt )

## Project structure 
Taskmgr/taskmgr.cpp , tasks.txt

# How it works

The Task Manager stores all task information in a local file named tasks.txt. Every operation performed through the CLI (adding, editing, deleting, or viewing tasks) updates or reads from this file, ensuring that tasks remain available even after closing the application.

## CRUD Operations

- Create 
- Read
- Update 
- Delete

# Building the Project

## Linux
`` g++ main.cpp -o taskmgr ``
``./taskmgr ``

## Windows
`` g++ main.cpp -o taskmgr.exe`` 
``taskmgr.exe``
